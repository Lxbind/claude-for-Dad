Lesson 05

Match tools to your work

~15 min practice questions then your own chart

Lesson 4 named the tools. This lesson teaches the judgment — looking at a task and picking the tool. The pattern, once you see it, is simple.

If Lesson 4's names felt slippery, this is where they lock in. You'll pick tools for made-up tasks first, then for your own — and your own is where the pattern really sticks. — L.

Reminder — here's what each surface actually looks like. Flip through if you need a refresher before the practice quizzes.

The decision rule

Ask three questions about a task, in order:

  1. Does it need to read files on my computer? → if yes, Claude Code.
  2. Does it need to touch other apps (email, calendar, internal systems)? → if yes, Claude Desktop + MCP.
  3. Otherwise → claude.ai is the answer.

Then a fourth question on top:

  1. Do I do this task more than once a week with the same shape? → wrap it in a skill so you don't retype.

Plus a fifth, situational:

  1. Do I keep re-pasting the same setup paragraph? → put it in a Project.

That's the entire mental model. Let's practice.

Practice 1 — A recurring summary

"Every Monday, I summarize the previous week's supplier interactions into a one-page brief for my department lead." Which tool fits best?

  • claude.ai — paste the notes, ask for a summary
  • A skill — same task, same shape, every week
  • Claude Code
  • MCP / Claude Desktop

Practice 2 — Reading 12 contracts

"I have 12 PDF contracts from a category. I need to extract the key terms (price, expiry, termination clause) into a comparison table." Which tool fits best?

  • claude.ai — upload them one by one and ask
  • Claude Code — point it at the folder of PDFs
  • A skill
  • MCP / Claude Desktop

Practice 3 — Checking the calendar

"I want Claude to check my Google Calendar, find a 30-minute slot next week that works for me and three suppliers, and draft the invite." Which tool fits best?

  • claude.ai
  • Claude Code
  • Claude Desktop + Google Calendar MCP server
  • A skill

Practice 4 — One-off question

"I want to draft a tough email to a supplier whose delivery is three weeks late. Just this once." Which tool fits best?

  • claude.ai — describe the situation, ask for a draft
  • A skill
  • Claude Code
  • MCP

Practice 5 — Repeated setup

"Every chat I open with Claude, I start with the same paragraph: 'I'm a procurement consultant at Sanofi advising internal departments...' Is there a better way?

  • Just keep pasting the paragraph
  • Create a Project in claude.ai with that paragraph as the instructions
  • Make it a skill
  • Set up an MCP server
